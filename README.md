# Phantom Dev Tool

## Sobre

Trata-se um projeto voltado a correção e validação de código server side.

## Tecnologias Utilizadas

- MongoDB
- Python
- FastApi

## Instalação

- Com o docker instalado basta rodar o comando `docker-compose build --no-cache && docker-compose up`
